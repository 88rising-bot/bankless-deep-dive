# 🎧 Bankless Deep Dive

> **Bankless播客深度笔记 · 中文加密学习知识库**

Bankless 是加密世界最前沿的播客之一，嘉宾包括 Vitalik Buterin、Michael Saylor、Raoul Pal 等顶级思想者。本仓库是我个人深度学习的产出——**不只是听，而是拆解、反推、沉淀框架**。

## 📂 结构

```
📁 episodes/       ← 逐期笔记（按时间归档）
📁 frameworks/     ← 抽出来的分析框架
📄 INDEX.md        ← 主题索引
📄 _TEMPLATE.md    ← 笔记模板
```

## 🧠 方法论

每期播客我采用 **C+D 双路线**：

1. **C — 反向演练**：听前写预判 → 暂停写分析 → 比对嘉宾逻辑 → 记差距
2. **D — 知识库沉淀**：用结构化模板归档，定期抽取框架
